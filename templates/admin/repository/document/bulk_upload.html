How it works

  • The filename will be used as the document title (formatted nicely)
  • Authors will be set to "Unknown" - you can edit each document later to add details
  • Files are uploaded and stored securely in the repository
  • Each document gets a unique filename to avoid conflicts
  • You can edit each document later to add more details like authors, abstract, etc.
